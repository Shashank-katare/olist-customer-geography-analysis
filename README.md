# Olist Customer Geography Analysis

## Project Overview
This project analyzes the geographic distribution of customers in the Olist e-commerce platform to identify regional concentration patterns across Brazilian states.

## Objective
- Analyze customer distribution by state
- Identify regions with the highest number of unique customers
- Demonstrate proper data validation and visualization practices using Python

## Dataset
- **olist_customers_dataset.csv**
  - Customer-level data including city and state information

## Tools & Libraries
- Python
- pandas
- matplotlib
- seaborn
- Jupyter Notebook

## Methodology
1. Loaded and inspected the customer dataset
2. Validated data quality (checked duplicates and null values)
3. Aggregated data using unique customer identifiers
4. Visualized the top 10 states by unique customer count
5. Interpreted results to derive business insights

## Key Insight
Customer distribution is highly concentrated in a small number of states, suggesting regional dominance and opportunities for targeted marketing and logistics optimization.

## Output
- Bar chart showing top 10 states by unique customers
- Clean, reproducible Jupyter Notebook
- Saved visualization image for portfolio use
